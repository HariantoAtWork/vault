Redesigned Bitwarden vault management UI from scratch with login, server selection, auth composable, and vault context switching.

## Added
- Login page with email, password, remember email, and server options (bitwarden.com, bitwarden.eu, self-host)
- Auth composable for BASE URL and token management via Nitro proxy routes
- Main vault UI with clear My Vault vs Organisation context, search/filter, and Bitwarden brand styling
- Crypto utilities for PBKDF2/Argon2 login hashing and cipher decryption after sync

## Why
The default Bitwarden web UI requires too many clicks to switch vaults and find passwords. This redesign makes vault context immediately visible and reduces navigation friction.
