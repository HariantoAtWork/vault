# Redesigned vault UI with Nuxt UI and Bitwarden brand theme

Applied frontend-design and Nuxt UI skills to improve usability and visual consistency across login and vault views.

- Configured Bitwarden brand colours as a custom `bitwarden` theme in Tailwind/Nuxt UI
- Refactored vault layout to `UDashboardGroup`, collapsible sidebar, navbar, and toolbar patterns
- Replaced raw HTML inputs/buttons with accessible Nuxt UI components
- Added copy toasts, password reveal, keyboard search shortcut (⌘K), and clearer empty/loading states
